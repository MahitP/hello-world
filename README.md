# Hello World (Java)

> A minimal Java application demonstrating basic program structure and execution flow.

---

## 📌 Overview

This repository contains a simple Java program that prints output to the console.  
It serves as an introductory exercise to Java syntax, class structure, and program execution.

While simple in functionality, the project follows clean code structure and standard Java conventions.

---

## 🧠 Key Concepts Demonstrated

- Java class definition
- Constructor usage
- Object instantiation (`new` keyword)
- Entry point (`main` method)
- Standard output using `System.out.println()`

---

## 📄 Code Summary

The program defines a `HelloWorld` class. When executed, it:

1. Instantiates the `HelloWorld` object
2. Triggers the constructor
3. Prints `"Hello World"` to the console

---

### Run the program:

### Prerequisites
